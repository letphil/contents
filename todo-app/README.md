# TODO App

## frontend[]

```
just simple full stack todo app. person wll be able to add Todo item
todo item will be [_id, task, createdAt, updatedAt, deletedAt]

frontend will have button to add todo list item
frontend will display in list format that update or delete that item

backend will have CRUD for TODO -> CREATE, READ, UPDATE, DELETE
endpoints ->
  /api/v1/todos/add-task
  /api/v1/todos/get-tasks
  /api/v1/todos/update-task
  /api/v1/todos/delete-task



will use react with vite
  - axios
```
